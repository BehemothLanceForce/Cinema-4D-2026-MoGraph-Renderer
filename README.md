## How It Works

Automated 3D studio setup:

- Downloads Cinema 4D with full plugin library
- Installs GPU compute drivers for Redshift
- Includes HDR environments and material presets
- Configures project defaults and output settings

## Common Issues

**MoGraph cloner slow?**
Use Instance mode. Reduce clone count.

**Simulation crashes?**
Increase cache: Project Settings → Dynamics → Cache → 4 GB+.
